# Customer-360-Churn-Analysis
End-to-end Customer 360 platform using SQL, Python, and Tableau to predict churn and protect revenue. Achieved 81% recall with a Random Forest model optimized via SMOTE and GridSearchCV. Identified $264K in revenue risk and pinpointed a 6–12 month "Danger Zone" for retention. Includes a dynamic priority call list for proactive stakeholder outreach.
